# -ZR-ChanceViewer
Output common chance of infection in HUD

## Requirements
 - [ZR](https://github.com/Franc1sco/sm-zombiereloaded-3-Franug-Edition)
 - *>=* SM 1.7

## Setup
1) Compile
2) Move in *plugins* folder
3) Load plugin or restart the server
